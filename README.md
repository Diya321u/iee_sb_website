# IEEE Student Branch STM Website

This repository contains the official website of the IEEE Student Branch, St. Thomas College of Engineering and Technology, Kannur.  
The website is a static informational platform developed to present details about IEEE, the Student Branch, its activities, leadership, and membership.

---

## Live Website

The website is publicly hosted using GitHub Pages and can be accessed at:

https://<diya321u>.github.io/ieee-sb-stm-website/

---

## Technologies Used

- HTML5  
- CSS3  
- GitHub Pages

---

## Website Sections

The website includes the following sections:

- Home  
- About IEEE  
- Societies & Affinity Groups  
- About IEEE Student Branch  
- Events & Activities (Academic Years 2023–24 and 2024–25)  
- Upcoming Events (2026)  
- Gallery  
- Executive Committee  
- Membership  
- Contact Us (Footer)

## Purpose of the Project

This project was developed as part of an academic requirement to demonstrate the design and deployment of a static website using fundamental web technologies.  
The website focuses on clear structure, accessibility, and professional presentation of information related to the IEEE Student Branch.
